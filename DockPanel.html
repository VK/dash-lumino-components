new DockPanel()

Description

A widget which provides a flexible docking area for widgets.
https://jupyterlab.github.io/lumino/widgets/classes/dockpanel.html

Examples
//Python:
			import dash
			import dash_lumino_components as dlc
			
			dock = dlc.DockPanel([
			    dlc.Widget(
			        "Example Content",
			        id="initial-widget",
			        title="Hallo",
			        icon="fa fa-folder-open",
			        closable=True)
			], id="dock-panel")
